# Explaining Time complexities with example :-

O(1) :- Like accessing an element from an array.
eg-
arr = [12, 9, 10, 27, 25, 30, 2]
print(arr[2])

O(n) :- Loop which is iterating n times.
eg-
for i in range(n):

O(n^2) :- Worst case scenario when using nested loop.
eg-
for i in range(n):
  for j in range(n):
