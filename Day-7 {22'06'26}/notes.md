A Series is a one-dimensional array-like object. You can think of it as a single column of data in an Excel spreadsheet.
* Dimensions: 1D (length only).
* Homogeneous Data: All elements in a Series typically share the same data type (e.g., all integers, all strings).
* Components: It consists of the actual data and a sequence of labels associated with it, called the index.



A DataFrame is a two-dimensional, size-mutated, tabular data structure. You can think of it as an entire Excel spreadsheet or a SQL table.
* Dimensions: 2D (rows and columns).
* Heterogeneous Data: Different columns can hold different data types (e.g., one column of strings, one column of integers, one column of floats).
* Structure: It aligns data in a spreadsheet-like structure with both a row index and column labels.
